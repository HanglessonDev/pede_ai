# Pede AI

Sistema de assistente virtual com RAG (Retrieval-Augmented Generation) para atendimento e extração de informações.

## 📚 Documentação

A documentação completa está disponível em: https://hanglessondev.github.io/pede_ai/

## 🚀 Funcionalidades

- Classificação de intenções com confidence score
- RAG para recuperação de contexto relevante
- Extração de entidades com spaCy
- Graph-based state machine para fluxo de conversa

## 🛠️ Instalação

```bash
uv sync
```

## 📝 Uso

```bash
uv run python main.py
```

## 🧪 Testes

```bash
uv run pytest
```
